# Поддержка языка QSP

Полнофункциональная поддержка языка QSP (Quest Soft Player) для Visual Studio Code, основанная на сервере Language Server Protocol (LSP) и грамматике [tree-sitter](https://tree-sitter.github.io/).

* Страница на GitHub: [https://github.com/QSPFoundation/qsp-lsp](https://github.com/QSPFoundation/qsp-lsp)
* Релизы на GitHub: [https://github.com/QSPFoundation/qsp-lsp/releases](https://github.com/QSPFoundation/qsp-lsp/releases)
